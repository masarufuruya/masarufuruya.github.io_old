rbenvを使って複数バージョンのRubyを切り替える

2014年01月18日

Tweet

以前の記事、MacOS MavericksにRails4 + Ruby2.0を構築する方法で、

ruby2.0.0-p0をインストールしたら、開発で利用していたresque-webが動かなくなったので

以前のバージョンのrubyにrbenvで戻す方法を調べた。

インストールされているrubyのバージョンを調べる

$ rbenv versions

以下のような結果が表示されます。

今選択されているバージョンには * がついています。

  system
  1.9.3-p194
* 2.0.0-p0 (set by ~/.rbenv/version)

rubyのバージョンを変更する

1.9.3-p194が戻したいrubyのバージョンなので、

以下のコマンドで元に戻します。

$ rbenv global 1.9.3-p194

変更を確認します

$ rbenv global

1.9.3-p194 と表示されればOK

インストール可能なrubyのバージョンを表示

$ rbenv install -l

そして、下記コマンドでインストールしたいバージョンを指定してインストールします。

$ rbenv install インストールしたいバージョンを指定

以上のようにすれば、任意のバージョンのrubyをインストールすることも出来ます。
