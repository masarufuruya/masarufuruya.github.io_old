最低限覚えておきたいSublime Text2のショートカットまとめ

2014年01月11日

Tweet

Sublime Text2とても便利ですよね。

ソフト自体は無料で利用することが出来ますが、エンジニアとしては素晴らしいツールに対しては

お金を払わなくては！ということでちゃんとお布施(約7000円)払って利用しています。

今回はSublime Text2を使い倒す上でこれだけは最低限押さえておきたいショートカットをまとめてみました

検索系

cmd + shift + f ファイル全体を検索

cmd + f 開いているファイル内を検索

cmd + p ファイル名で全体を検索

移動系

ctrl + g 指定の行へ移動

cmd + r 指定のメソッドへ移動

cms + ←↑⇛↓ 端へ移動

cms + shift + p コマンド入力

alt + カーソル 単語単位の移動

コピー・削除

cmd + c (行の操作) 選択しているものがあればそれをコピーする。　何も選択していない状態では行全体をコピーする。

control + shift + k 行の削除

cmd + shift + d 現在の行の複製

マルチカーソル

control + shift + ↑↓ マルチカーソル

cmd + d 単語の選択(選択された状態でcmd + dを押すとマルチカーソル)

cmd + k, cmd + d マルチカーソルでスキップ

cmd + u マルチカーソルで前に戻る

特に自分がよく使うもの

cmd + p ファイル名で全体を検索

ctrl + g 指定の行へ移動

cmd + r 指定のメソッドへ移動

cmd + shift + ↑↓ マルチカーソル

cmd + d 単語の選択(選択された状態でcmd + dを押すと同じ単語を複数選択)

次はZen-Cordingのショートカットについてまとめたいと思います！

ではまた次回！
