Mongo DBの基本的なコマンドまとめ

2014年02月02日

Tweet

先週、日本に帰国してからバタバタしていてブログが疎かになっていたので

今日から引き続き、更新していきます。

よく使うMongo DBのコマンドをまとめました。

起動 & 終了

起動

$ mongo

終了

> exit

データベースの操作

DBの一覧表示

> show dbs;
test (empty)
localhost (empty)

DBの選択 & 作成

> use [DB名]

MongoDBの場合は、SQLのように明示的にcreateする必要はありません。

useコマンドで使いたいDBを指定して、その環境にそのDBが存在しなければ、 自動的に作成されそのDBが指定されます。

既に同じ名前のDBが存在すれば、そのままスイッチします。

DBの削除

> db.dropDatabase();

コレクションの操作

コレクションの生成

> db.createCollection('[コレクション名]');

コレクションの一覧表示

> show collections

コレクションの削除

> db.[コレクション名].drop();

コレクション名の変更

> db.[コレクション名].renameCollection('[新コレクション名]');

データの操作

データの挿入

> db.[コレクション名].insert({“name”:"test", “email”:”test@gmail.com”, “age”:”25″});

データの更新

// update [コレクション名] set y = 1 where x = 1
db.[コレクション名].update({ x:1 }, { $set: { y : 1 } });

データの削除

> db.[コレクション名].remove();

検索

一覧表示

> db.[コレクション名].find();

条件付きの検索(年齢が20歳のものだけ)

> db.[コレクション名].find({“age”:20});

条件付きの検索(年齢が20歳以外のもの)

※ $ne はnot equal

> db.[コレクション名].find({“age”:{$ne:20}});

条件付きの検索(年齢が20歳より上のもの)

※ $gt　はgreater than

> db.[コレクション名].find({“age”:{$gt:20}});

インデックスの作成

db.[コレクション名].ensureIndex({id:1});

複合インデックス

db.[コレクション名].ensureIndex({x:1, y:1});

※インデックスを貼ると参照は早くなりますが、データを書き込む時には都度インデックスを貼り直すので遅くなります。
