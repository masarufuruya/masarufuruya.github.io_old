Vagrant入門！コマンドラインからLinux環境をセットアップしよう

2014年02月12日

Tweet

Vagrantをインストール

各OSにあった最新版のインストーラーをダウンロードする

http://downloads.vagrantup.com/

ダウンロードしたインストーラーの指示に従ってインストール

インストールが完了するとターミナルからvagrantコマンドが利用出来るようになります。

プロジェクトフォルダの作成

$ mkdir TestApp
$ cd TestApp
$ vagrant init

Boxファイルをローカルのvagrantプロジェクトに追加

$ vagrant box add test https://github.com/2creatives/vagrant-centos/releases/download/v0.1.0/centos64-x86_64-20131030.box

※上記リンクがリンク切れの場合は、このサイトから各OSのboxファイルをダウンロード出来ます。

http://www.vagrantbox.es/

追加したBoxファイルを読み込むように設定

$ vim Vagrantfile

以下のように修正

Vagrant.configure("2") do |config|
    config.vm.box = "test"

VirtualBoxを起動する

$ vagrant up

動作確認を行うためSSHでログインする

$ vagrant ssh

ホストマシンとゲストマシンのファイルを共有する

Vagrantfileを下記の通り、修正

config.vm.synced_folder "./host_synced", "/vagrant/host_synced"

設定を反映するため、再起動

$ vagrant reload

VMのguest Addtionのバージョンが古いというエラーが出てくる場合は、

下記サイトのプラグインをインストールして最新版にアップデートします

http://blog.csanchez.org/2012/05/03/automatically-download-and-install-virtualbox-guest-additions-in-vagrant/

インストール

vagrant plugin install vagrant-vbguest

今回はインターネットで公開されているboxファイルから、

CentOSをインストールする方法のみの紹介になりましたが、

自分用にカスタマイズしたboxファイルを用意することも出来ます。

機会があれば、また次回！
